# media-tools

Client-ready media prep skills.

- **video-to-gif** — convert a video, or a chosen time range of it, into a balanced-quality animated GIF (ffmpeg, two-pass palette).
- **pptx-to-pdf** — convert a presentation to a high-fidelity PDF with LibreOffice: embedded fonts, layout preserved, and an automatic page-count check that warns if a slide was dropped.
- **compress-pdf** — shrink a PDF under an email size limit (Ghostscript), downsampling images while keeping fonts/text intact; never returns a file larger than the original.
