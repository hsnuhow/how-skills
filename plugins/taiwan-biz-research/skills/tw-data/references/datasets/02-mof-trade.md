## 一、枚舉方法（已驗證可用）

data.gov.tw 是 Nuxt SPA，沒有公開的「機關 → 資料集清單」REST API，但前端搜尋走一支可直接重放的內部 API。逆向其 JS bundle（`/_nuxt/Doi2eyNx.js`）後確認：

**枚舉引擎** — `POST https://data.gov.tw/api/front/dataset/list`（需瀏覽器 UA + `Referer: https://data.gov.tw/`，`Content-Type: application/json`）。Body schema：

```json
{"bool":[ {"fulltext":{"value":"綜稅"}} ],
 "filter":[], "page_num":1, "page_limit":100, "tids":[], "sort":"_score_desc"}
```

- **關鍵字搜尋**：`bool:[{"fulltext":{"value":"關鍵字"}}]`。（注意：頂層放 `qs`/`keyword` 都被忽略、回整個目錄 53,207 筆——關鍵字**必須**包在 `bool[].fulltext.value`。）
- **依機關窮舉**：`bool:[{"agency_tid":{"value":438}}]` → 該機關全部資料集。此為列出「某機關所有 dataset」的正解。
- `page_limit` 實測可到 1000；分頁靠 `page_num`。回傳 `payload.search_count` + `search_result[]`，每筆含 `nid, title, agency_name, agency_tid, central_government_level_2, updatefreq_desc, all_file_format_name, all_url, coverage_temporal_*` 等完整 metadata。
- **`nid` == 公開 datasetId**：`nid=144533` → `https://data.gov.tw/dataset/144533` 且 `https://data.gov.tw/api/v2/rest/dataset/144533`（本次驗證一致）。單筆 metadata 用 v2 REST（需瀏覽器 UA；urllib 403）取 `distribution[].resourceDownloadUrl` 拿真實下載 URL。

**機關樹** — `GET /api/front/agency/listbytid/432` 回財政部（tid **432**）底下所有子機關 tid 陣列。關鍵 agency_tid：

| tid | 機關 | 資料集數 | 下載主機 | 健康 |

| **438** | 財政部財政資訊中心（FIA） | 891 | www.fia.gov.tw | ✅ 活 |
| 434 | 財政部統計處 | 56 | web02.mof.gov.tw / service.mof.gov.tw | ⚠️ 見卡點 |
| 436 | 財政部關務署 | 166 | opendata.customs.gov.tw | ✅ 活 |

## 二、核心：綜合所得稅申報（消費力研究最高優先）

FIA（438）出的「綜稅」系列共 **746 筆**，是全體申報戶（非抽樣），逐年一 CSV 檔（`www.fia.gov.tw/WEB/fia/ias/ias{ROC}/{ROC}_{表號}.csv`），健康。**每個主題都有五種切法**：各縣市 / 各級距 / 5 分位 / 10 分位 / 20 分位。此結構直接給出「各地區 × 各所得層級」分布，正是消費力分層的骨幹。

**特別點名（村里級，最細地理粒度）**：
- **103066 綜稅綜合所得總額全國各縣市鄉鎮村里統計分析表** — 細到村里，年更，CSV，`www.fia.gov.tw`。全台最細的所得地理分布公開資料，遠比家庭收支調查（縣市/分位）細。**消費力研究首選**。

## 六、卡點（誠實標注）

1. **web02.mof.gov.tw/njswww 匯出當前不通**：主機根路徑回 200（活著），但 `webMain.aspx` 匯出端點本次連 `mof_industry.py` 已知可用的營業稅 URL 都 **60 秒 timeout（curl code 000）**，重試兩次皆敗。受影響：統計處全部**貿易 HS/貨品別×國別、貿易指數、全國賦稅收入實徵淨額**系列（28573、28536-58、6671/6741/6742、8378、8387-89 等）。這批的 data.gov.tw resource URL 全指向此主機。tw-data 記錄它「偶爾斷線、重試一次」，但此刻是持續性阻斷——研判為環境端網路/節流問題（非台灣 IP 常被擋），需改 browser skill、台灣出口節點、或稍後重試。**這是唯一真正阻斷的一批，且它涵蓋「貨品別 HS×國別貿易」與「賦稅統計」兩個題目要求的重點，必須標紅。**替代：貿易走 6053(customs)＋15392(service.mof)＋國貿署 FSC 重放；賦稅走 web02 重試或財政部統計月報 PDF。
2. **POST /dataset/list 依機關過濾只吃 `agency_tid`（子機關），不吃 `central_government_level_2`（部會層 432）**——回 0。要窮舉整個財政部須先 `listbytid/432` 取子機關 tid 逐一過濾（已完成）。
3. FIA 逐年 CSV 檔名年份相關（`{ROC}_{表號}.csv`），新舊系列表號不同（舊 `_6-1`、加計股利 `_6A-1`）；抓取時務必讀 v2 REST 的 `distribution[]` 拿實際各年檔 URL，勿自行拼年份（試 `112_6-1.csv` 回 404）。
4. 綜稅口徑限制：營業稅法第8條使部分免稅；分開計稅股利自107年改制，跨年比較須用「加計股利」新系列避免斷點。
