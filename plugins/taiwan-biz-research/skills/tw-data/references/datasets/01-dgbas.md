# 行政院主計總處（DGBAS）data.gov.tw 資料集索引

## 一、枚舉方法（可複用於任何機關）

`data.gov.tw` 是 Nuxt SPA + Nitro 後端，沒有公開的「列出某機關所有 dataset」的 REST 端點。實測結論：

| 端點 | 結果 |
|---|---|
| `GET /api/v2/rest/dataset/{numeric_id}` | **可用、免金鑰**。但 `{id}` 必須是**數字**（`identifier` A41… 那種是另一欄，非路由鍵）。要帶瀏覽器 UA，Python urllib 會 403；curl `-A` 或 WebFetch 可。 |
| `GET /api/v2/rest/dataset?organizationId=X` | 405 Method Not Allowed |
| `POST /api/v2/rest/dataset` | 200 但回 `API Key錯誤：HTTP 標頭沒設定 Authorization Key` → **列表 API 需要申請的 Authorization Key**，匿名不可用 |
| `/api/v2/rest/organization` | 404 |
| `/sitemap.xml`、各種 `/api/front/...` `/api/v1/...` | 404/500，前端 API 路徑未公開 |

**可行且穩健的枚舉法 → 用平臺自己發布的「全站資料集清單」資料集**：

1. 平臺把整站 catalog 當成 dataset 發布：**dataset 6564「政府資料開放平臺資料集清單」**。
2. 其下載點是固定匯出端點，無需金鑰、無需分頁：
   - CSV：`https://data.gov.tw/datasets/export/csv`（本次抓到 **68 MB，74,417 筆**全站資料集）
   - 另有 `/datasets/export/xml`、`/datasets/export/json`
3. CSV 欄位含 `提供機關`（第 12 欄）、`資料集識別碼`、`資料集名稱`、`檔案格式`、`資料下載網址`、`更新頻率`、`資料集描述` 等。**用 `提供機關 == "行政院主計總處"` 精確過濾即得該機關全部 dataset**（本法對任一機關通用，改機關名即可）。
4. 逐筆核對用 `GET /api/v2/rest/dataset/{id}`（含 `distribution[].resourceDownloadUrl`、`coverageStartedDate/EndedDate`）。注意：**coverage 欄在 DGBAS 資料集多為空字串**，實際涵蓋期間要看檔案內容。

> 提醒其他 agent：不要試圖用 `organizationId` 或猜前端 API，會卡在 405/需金鑰。直接抓 catalog CSV 過濾最快。

## 四、誠實標注枚舉/覆蓋缺口

- **1,945 筆已全部枚舉**（catalog CSV 過濾），本報告逐筆列出的是高/中相關的旗艦表；503 筆中央政府預決算、258 筆產業關聯、221 筆農林漁牧普查、344 筆雜項屬低相關，已給總數與代表樣本，未逐一列 id（清單在 `/tmp/dgbas.json`，可全量交付）。
- **資金流量表（Flow of Funds）不在 DGBAS**：全站以「資金流量」為名的 DGBAS 資料集為 0 筆——此表由**中央銀行**發布，研究時要改接央行來源。
- **超額儲蓄**無獨立 dataset，需由國民所得統計 44231/44232（儲蓄與投資）自行計算。
- **coverage 期間**：API 的 `coverageStartedDate/EndedDate` 對 DGBAS 幾乎全空，涵蓋期間需開檔判讀（已抽驗：CPI 6019 自 1981M01）。
- 「性別統計」在平臺分類下多被農漁普查的性別細分表佔據；家庭收支調查自己的性別統計指標表（如 130026、91919、144913）才是消費力相關的性別資料，已列於 A 區。
