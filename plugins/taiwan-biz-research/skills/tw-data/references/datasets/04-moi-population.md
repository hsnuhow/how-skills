# data.gov.tw 內政部（MOI）人口與區域統計資料集索引

## 一、枚舉方法（實測可用）

**A. data.gov.tw 單一 metadata API（可用，需瀏覽器 UA）**
- `GET https://data.gov.tw/api/v2/rest/dataset/{id}` — `curl -A "Mozilla/5.0"` 或 WebFetch 皆可，回 JSON。含 `title / dataProvider / publisherOID / updateFrequency / distribution[].resourceDownloadUrl`。
- 用 `publisherOID` 判機關歸屬：`2.16.886.101.20003.20001.20033` = **內政部戶政司**（provider 帳號 `bannanng30`）；`...20069` = 國發會 NDC；`...20071` = 主計總處 DGBAS。這點很重要——很多「人口」資料集掛在 data.gov.tw 但其實是 NDC/DGBAS，不是戶政司。

**B. organizationId 枚舉（卡點，未突破）**
- `GET .../rest/dataset?organizationId=X` → 405；改 **POST → 需 Authorization Key**（回 `ER0001:API Key錯誤`）。無金鑰無法批次列機關資料集。前台 `data.gov.tw/api/front/dataset/search` → 404。**繞道方式**：改用下述 RIS docs 頁一次列出戶政司全系列，再用 WebSearch 補齊統計處/NDC/DGBAS 的零星 id。

**C. 戶政司自有 API（最關鍵，全序列可枚舉）**
- 枚舉入口：`https://www.ris.gov.tw/rs-opendata/api/Main/docs/v1`（HTML docs 頁，一次列出全部 ODRP 編號＋中文名）。用 `grep -oE "ODRP[0-9]{3}"` 得到完整清單：**ODRP001–072、094、115–118**。
- 取數/下載端點：`GET https://www.ris.gov.tw/rs-opendata/api/v1/datastore/{ODRP編號}/{期間}`
  - **年度資料 → 期間用民國年 `yyy`**（例：`/ODRP052/113`）
  - **月報/動態/登記類 → 期間用民國年月 `yyymm`**（例：`/ODRP061/11312`）
  - 回 JSON，含 `totalPage/totalDataSize`，每頁 2000 筆，`?page=N` 翻頁。**此端點本身即下載來源**，data.gov.tw 上戶政司資料集的 `resourceDownloadUrl` 指向 `opdadm.moi.gov.tw`（同一套）。
- data.gov.tw id ↔ ODRP 對照（已確認）：**8409=ODRP020、8410=ODRP048、32973=ODRP005/村里單一年齡**。其餘 ODRP 皆可直接用 RIS API，不必先查 data.gov.tw id。
- **重要規律**：舊版（純號、非「含同婚」、「新增區域代碼」前身）多已停更，`/113` 回「查無資料」（ODRP004/005/013/014/015/021/023 等）；**現行維護中的是「含同婚」與高編號版本**（ODRP052/053/054/060/061 等）。接資料要接新版。

**卡點誠實標注**：(1) organizationId 批次列表需 API Key，未取得；(2) `openDataDescription` 端點回 404，各資料集的「正確期間格式與最早涵蓋年」需逐一試 `yyy` vs `yyymm`＋往前試年份確認；(3) ODRP034/041/062/063 等以 `/113` 查無，須改月別或特定期間（ODRP062 已確認 `/11312` 可用）。

## 三、對區域消費力分層最該優先接的 8 個

1. **ODRP052 現住人口性別/年齡/婚姻(含同婚)** — 里×單一年齡×性別×婚姻。分層地基，可自算任何年齡結構指標。`/113`✅
2. **ODRP053 15歲以上人口按性別/年齡/婚姻/教育(含同婚)** — 里×齡×性×**教育程度**。教育＝消費力最強代理變數，且到里級。`/113`✅(1367萬筆)
3. **ODRP048 / id 8410 各鄉鎮市區人口密度** — 區×(人口/面積/密度)。城鄉分層、通路密度。`/113`✅
4. **ODRP019 戶數、人口數按戶別及性別** — 里×戶別×性。戶數/戶量→家戶消費單位。`/113`✅
5. **ODRP060 各村里人口統計月報表(含同婚)** — 里×性、**月更**。最新人口/戶數即時面。`/11312`✅
6. **ODRP020 / id 8409 各村里教育程度資料** — 里×教育，欄位精簡好接，適合快速分層原型。`/113`✅
7. **NDC id 6327（年齡結構及扶養比例）** — 縣/主要都市級，**現成扶養比/老化**，省去自算，適合縣市層速覽（下載主機 ws.ndc.gov.tw）。
8. **ODRP061 動態統計表(含同婚)** — 里×性，出生/死亡/**遷入遷出**合一、月更。抓區域人口動能（移入熱區＝消費潛力上升）。`/11312`✅

補充：新住民消費力可接 **ODRP072（初設戶籍）+ ODRP041–044（歸化）+ ODRP028/059（生母國籍別出生）**；原住民接 **ODRP054**。
