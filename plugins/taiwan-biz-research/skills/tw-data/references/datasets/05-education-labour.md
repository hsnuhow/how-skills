# data.gov.tw 教育 × 勞動力 資料集索引（供台灣消費力研究工具）

## 一、枚舉方法（已實測，2026-07-18）

data.gov.tw **沒有可用的關鍵字搜尋 API**，但平台把「全站資料集清單」自己發布成一個資料集，這就是完整枚舉的鑰匙。

| 機制 | 結果 | 用法 |
|---|---|---|
| **全站目錄 = dataset 6564「政府資料開放平臺資料集清單」** | ✅ **可用（最佳解）** | 下載 `https://data.gov.tw/datasets/export/csv`（瀏覽器 UA），68 MB，**53,208 個資料集**，22 欄含 id／名稱／格式／下載網址／機關／更新頻率／描述。每日 01:00 更新。 |
| 單筆 metadata `GET /api/v2/rest/dataset/{id}` | ✅ 可用 | 需瀏覽器 UA（curl -A）。回傳 coverage 日期、真實下載 URL、欄位清單。無效 id 回 `{"success":false,...Not Found}`。 |
| `GET /api/v2/rest/dataset?q=` / `?organizationId=` | ❌ 405 Method not allowed | 此路徑只吃 `/{id}` |
| `POST /api/v2/rest/dataset` | ❌ 需 `Authorization Key`（M2M 金鑰，須註冊） | 無金鑰不可用 |
| organization API | ❌ 404 | — |
| `/api/front/...`（前端搜尋/統計匯出） | ❌ 需授權金鑰 | — |
| **勞動部自有 API** `GET https://apiservice.mol.gov.tw/OdService/rest/dataset` | ✅ 可用 | 回 **859 個 datasetId**（非 860）；`/dataset/{id}` 給完整 metadata + 每個 distribution 的下載 URL（`.../OdService/download/{resourceID}`）。勞動部家族的資料在 data.gov.tw 上的下載主機也都是 `apiservice.mol.gov.tw`，兩邊一致。 |

**建議工作流**：以 dataset 6564 的 CSV 為主索引（一次抓完、離線過濾）→ 需要 coverage/欄位細節時再打 `/api/v2/rest/dataset/{id}` 單筆補。勞動部資料可改用 MOL API（不需 UA 偽裝、較穩）。
已下載的完整目錄留存於 `/tmp/datagovtw_catalog.csv`。

## 六、卡點與誠實標注

- **無關鍵字搜尋 API**：data.gov.tw 前端搜尋與 `POST /dataset` 都要 M2M 授權金鑰（需註冊申請）。枚舉只能靠 dataset 6564 全量 CSV 離線過濾——可行且完整，但每次是 68 MB 全量下載。
- **coverage 日期常為空**：多數 metadata 的 `coverageStartedDate/EndedDate` 是空字串（如 40121、14435），實際期間要下載 CSV 看內容才知道（已抽驗：40121 到最新學年、14435 到 113年）。
- **格式不一**：教育部統計處＝乾淨 CSV+JSON（好接）；主計總處＝多為 **XML**（`ws.dgbas.gov.tw`，家庭收支調查是 CSV），需 XML parser；勞動部＝CSV/JSON/XML/WEBSERVICES 齊全。
- **顆粒度天花板**：縣市級以下（鄉鎮市區）教育程度／薪資資料幾乎不存在；學校**名錄**（6240 等含地址）是唯一能 geocode 到鄉鎮市區的教育資料來源，需自建 geocoding。縣市級教育程度只有 10 年一次的普查（20729–20750）。
- **雜訊比高**：教育部 324 筆中大半是營養午餐、得獎名單、學術名詞；勞保局數百筆是「積欠工資墊償基金會計月報」。過濾務必用機關×精確關鍵字並排除「墊償/會計月報/決算/預算/名單/名詞」。
- **勞動部 API 回 859（非題目說的 860）** 個 datasetId；差異可能來自平台即時增刪，不影響使用。
- 未遇到 API 過載（此次全部走目錄 CSV + 單筆驗證，未大量打單筆端點）。
