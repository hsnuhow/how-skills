# data.gov.tw 經濟部（MOEA）及所屬資料集枚舉索引

## 一、枚舉方法（已實測，這是關鍵發現）

**可靠的窮舉法 = 下載平臺自身的「全量詮釋資料 CSV」，再按「提供機關」欄離線過濾。** 不需要 API Key。

| 方法 | 端點 | 狀態 | 用途 |
|---|---|---|---|
| **全量清單 CSV**（推薦） | `https://data.gov.tw/datasets/export/csv` | ✅ 可用（68 MB，53,208 筆，每日 01:00 更新） | 一次抓全站，離線用 `提供機關` 欄過濾出經濟部家族。這是資料集 **6564「政府資料開放平臺資料集清單」** 的 distribution。 |
| 全量清單 XML | `https://data.gov.tw/datasets/export/xml` | ✅ 可用（較慢） | 同上，欄位更完整 |
| 單筆 metadata | `GET /api/v2/rest/dataset/{id}`（需瀏覽器 UA） | ✅ 可用，**無需金鑰** | 取單集詳情：distribution 下載網址、coverageStartedDate、更新頻率、聯絡人 |
| organization / 依機關列表 | `/api/v2/rest/dataset?organizationId=X`、POST `/api/v2/rest/dataset` | ❌ GET 405、POST 回 `ER0001:API Key錯誤`（`Authorization` 標頭沒設） | **被金鑰擋死**，不能用來枚舉 |
| 前台搜尋頁 | `data.gov.tw/datasets/search` | ❌ Nuxt SPA，client-side render，WebFetch 只看到「無資料」 | 不可爬 |

CSV 欄位含：`資料集識別碼, 資料集名稱, 服務分類, 檔案格式, 資料下載網址, 資料集描述, 主要欄位說明, 提供機關, 更新頻率, 授權方式, 上架日期, 詮釋資料更新時間, 資料量` 等 22 欄。**注意 CSV 沒有「涵蓋期間」欄**——涵蓋期間須逐集打 `/api/v2/rest/dataset/{id}` 讀 `coverageStartedDate/EndedDate`（且常為空），或直接看下載檔內容。

**機關對照小發現**：`identifier` 前綴編機關（統計處＝`313000000G-xxxxxx`）；但 CSV 的 `提供機關` 欄才是可過濾鍵。統計處的資料集在 `提供機關` 欄一律掛「**經濟部**」（母部），不是獨立機關名。

## 五、卡點與誠實標注

- **`商業動態統計調查`／`電子商務`／「營業額按通路」在 data.gov.tw 上查無以此命名的 MOEA 開放資料集**。該調查是統計處的**調查報告/新聞稿**（發布於 moea.gov.tw 專頁與 EE110 系統），未以結構化 dataset 上架平臺。若需通路別/電商營業額，需改走統計處網站或 `service.moea.gov.tw/EE521` 視覺化系統，不在本平臺枚舉範圍。
- **依機關列出 API 全被金鑰擋死**（organizationId 405、POST 回 API Key 錯誤）；唯一可行窮舉法是全量 CSV 離線過濾，已採用。
- **涵蓋期間欄位缺失**：CSV 無涵蓋期間；`/api/v2/rest/dataset/{id}` 的 `coverageStartedDate` 多為空（6842、16365 皆空），實際起始須看下載檔（6842 實測自民國88年）。
- **更新頻率標示不可盡信**：多數旗艦統計 CSV 標「不定期更新」，但實際為月更（regularupdate 代碼 2＝不定期，但內容按月堆疊）。
- **下載主機分散**：統計類在 `service.moea.gov.tw/EE520` 與 `www.moea.gov.tw`；登記類在 `data.gcis.nat.gov.tw`；能源在 `www.moeaea.gov.tw`；中小企業在 `www.sme.gov.tw`。`service.moea.gov.tw` 對亂路徑會回 403，但正確 distribution URL（如 `/EE520/opendata/ea.csv`）回 200。

**產物檔**：全量清單已存於 `/tmp/datagovtw_all.csv`（68MB），過濾結果於 `/tmp/moea_sel.json`，驗證樣本 `/tmp/d6842.csv`。經濟部家族共 **2,659** 集，其中消費相關（高/中）約 **1,000** 集（高關聯 204、中關聯 796）。
