### 存取模式與失敗模式（四檔共通）

| 存取路徑 | 結果 |
|---|---|
| `https://service.moea.gov.tw/`（HTML 入口） | **HTTP 403**（Cloudflare），帶瀏覽器 UA 也 403 |
| `https://service.moea.gov.tw/EE520/opendata/`（目錄列表） | **HTTP 403** |
| `.../EE520/opendata/{中文檔名}.csv` | **HTTP 200**，`content-type: application/octet-stream`，`server: cloudflare` |
| CSV 不帶 User-Agent | **HTTP 200**（此 host 不做 UA 過濾） |
| `data.gov.tw/api/v2/rest/dataset/{id}` 不帶 UA | **HTTP 200**（此端點也不需 UA，且免 API key） |

- 入口 403 但檔案 200 —— **入口被擋不代表資料被擋**，不要因為 portal 403 就判定來源死亡。
- 檔名為**中文字元**，必須 percent-encode 路徑；Python `urllib` 對未編碼的非 ASCII URL 會拋 `UnicodeEncodeError`（`_fetch.py::encode_url` 處理此事）。
- 檔名含**全形頓號「、」**（16365），編碼後為 `%E3%80%81`，手打極易錯。建議一律經 `resolve_dataset(id)` 取 `resourceDownloadUrl`，不要硬寫 URL。
- **data.gov.tw metadata 的 `modifiedDate` 不可信**：16365／16366 都標 2025-06-12，實際檔案是 2026-06/07 更新、含 2026-05 資料。判斷新鮮度只能看 CSV 內 `資料期(民國年)` 最大值。

- 四檔皆為 long format（一 row = 一分類 × 一期），必須 pivot；且母項與子項同欄，**分類過濾一律用精確相等，禁止子字串比對**。
